# SCC-Pytorch

This is a Pytorch implementation of paper: Improving Unsupervised Image Clustering with Spatial Consistency. 

## Requirements

- python 3.6
- pytorch 1.7.1
- torchvision 0.8.2
- tensorboard 2.0.0
- numpy
- tqdm
- faiss
- scipy
- scikit-learn
- pyyaml
- easydict
- termcolor
- apex
